Easy Installation Guide

Setup the extension in under 60 seconds with these simple steps.

1

Download the ZIP File

Click the download button at the top to grab the extension source code package directly from GitHub.

2

Extract the Contents

Extract the downloaded ZIP folder to a convenient location on your computer (e.g., Desktop or Documents).

3

Enable Developer Mode in Chrome

Go to your Chrome browser and navigate to chrome://extensions/. Toggle the "Developer mode" switch in the top-right corner to ON.

4

Load the Unpacked Folder

Click "Load unpacked" in the top-left corner, and select the folder where you extracted the extension. It will load instantly!
