## 🛠️ Installation Guide

Since this extension is in development/unpacked format, follow these steps to load it into your Google Chrome browser:

1.  **Clone / Download the Repository:**
    ```bash
    git clone git@github.com:ImranGurmani/heygen-unlimited-genxempire.com.git
    ```
    *(Or download the ZIP file and extract it to a folder).*

2.  **Open Chrome Extensions Page:**
    In your Chrome browser URL bar, type:
    ```text
    chrome://extensions/
    ```

3.  **Enable Developer Mode:**
    Toggle the **"Developer mode"** switch on the top-right corner.

4.  **Load Unpacked Extension:**
    Click the **"Load unpacked"** button on the top-left and select the folder containing these files (where `manifest.json` is located).

5.  **Pin for Quick Access:**
    Click the puzzle piece icon on your Chrome toolbar and pin **GenXEmpire HeyGen** for easy access.

## 🚀 How to Use

1.  Generate your video on the **HeyGen** platform.
2.  Play the video preview or the generated video for a few seconds.
3.  Click the **GenXEmpire HeyGen** icon in your toolbar.
4.  Click **"Scan for Media"**.
5.  Click **"Download"** next to the detected HeyGen video file.
